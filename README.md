# FoggyVault

FoggyVault is a retro horror game archival and recommendation platform focused on older titles and obscure releases.

The project combines game preservation, metadata organization, and personalized discovery into a searchable platform for classic horror games.

Inspired by platforms like Letterboxd and IMDb, FoggyVault allows users to browse games, explore detailed game pages, save titles to a personal backlog, and discover similar games through atmosphere, themes, mechanics, and aesthetic similarities.

---

# Planned Features

- Retro horror game archive
- Detailed game pages
- Similar game recommendations
- “Want to Play” backlog system
- Search and filtering
- Atmosphere and aesthetic tagging
- Regional release information
- Hardware and emulation notes
- Obscure game discovery
- User accounts and authentication

---

# Recommendation System

FoggyVault recommendations may consider:

- genre similarities
- gameplay mechanics
- atmosphere and aesthetic tags
- console generation
- themes and visual style

Example tags:

- psychological horror
- survival horror
- foggy
- gothic
- eerie
- abandoned
- cult classic
- low-poly
